# 상품 등록 관리 페이지

## 프로젝트 소개

- typescript로 안정적인 개발 환경 구현

- json-server를 이용한 mock rest-api server 연동

- 에이팀벤처스의 파트너 요청 페이지, 필터 같은 옵션에 따라 해당하는 아이템만 받아올 수 있는 페이지 구현

- UI 라이브러리를 사용하지 않고 직접 구현

- 미디어 쿼리로 모바일에서도 가독성이 높은 반응형 페이지 구현
